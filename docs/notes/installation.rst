⬇️ Installation
================================


From pip
------------------------------------
You can install or upgrade AnglE from pip:

.. code-block:: bash

    python -m pip install -U angle-emb



From Github
------------------------------------
You can also install the latest AnglE from Github:

.. code-block:: bash

    git clone https://github.com/SeanLee97/AnglE.git
    cd AnglE
    python setup.py install
